- Troubleshooting & Documentation

## My Notes
- Azure resources can stack small charges when left running.
- If not setup properly, errors can occur during deployment easily
- Resource groups simplify management and permissions while organizing your cloud assets
- Learned that navigating Azure's UI can be tricky and correctly configuring resource and region configurations is critical
